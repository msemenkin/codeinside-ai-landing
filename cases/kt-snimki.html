Автоматизация разметки КТ-снимков

Интеллектуальная система для стоматологии на основе ML

О проекте

Разработка веб-приложения для автоматизации процесса разметки КТ-снимков в стоматологии с использованием технологий машинного обучения.

Задачи проекта

  • Автоматизация разметки КТ-снимков
  • Создание интуитивного интерфейса
  • Интеграция ML-моделей
  • Формирование отчетов об отклонениях

Технологии

Java Machine Learning Spring Boot React

Реализация проекта

1

Загрузка и обработка снимков

Система принимает КТ-снимки в различных форматах и подготавливает их для анализа ML-моделью

2

Автоматическая разметка

ML-модель анализирует снимки и автоматически размечает ключевые точки и области интереса

3

Проверка специалистом

Врач проверяет и при необходимости корректирует результаты автоматической разметки

4

Формирование отчета

Система генерирует подробный отчет о состоянии и возможных отклонениях

Результаты

Скорость обработки

Сокращение времени обработки КТ-снимков в 4 раза

Точность

Высокая точность автоматической разметки благодаря ML

Удобство

Интуитивный интерфейс для врачей-стоматологов
